# My Clicks - Photography Portfolio

This is the photography portfolio subdomain for amitadhikary.com, showcasing my photography work.

## Features

- Modern, responsive design
- Consistent branding with main portfolio
- Framer Motion animations
- Tailwind CSS styling

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm start
```

Runs the app in development mode at [http://localhost:3000](http://localhost:3000).

### Build

```bash
npm run build
```

Builds the app for production to the `build` folder.

## Deployment

This site is designed to be deployed at photos.amitadhikary.com as a subdomain.

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React Icons
